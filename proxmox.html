¿Qué es Proxmox?

Proxmox es un entorno de virtualización de servidor de código abierto el cual nos permite el despliegue y la gestión de máquinas virtuales y contenedores.

Instalación de Proxmox en el servidor

Se crea un pendrive booteable con Proxmox y se inicia el servidor con este conectado. No funcionaba bien a la hora de empezar a montarse. Por lo cual, hemos recurrido a hacerlo con una disquetera externa y un CD.

Al acabar la instalación e iniciar el servidor, nos conectamos con el usuario root y la contraseña que le hemos puesto.

Iniciamos sesión en Ubuntu (desde nuestro ordenador) y hacemos:

  apt-get upgrade
  apt-get update

Iniciamos sesión en el servidor y hacemos:

  apt-get upgrade
  apt-get update

Estos último pasos son simplemente para tener tanto el equipo como el servidor actualizados y al día.

Probamos a hacer ping desde el servidor a una IP (8.8.8.8) por ejemplo para comprobar que está bien conectado a la red.
Luego, hacemos ping desde el equipo a la puerta de enlace del servidor (10.9.53.113 en nuestro caso) para ver que estén conectados.

Entramos en el Proxmox desde el navegador escribiendo la IP del servidor y el puerto por el que nos conectaremos.

Instalaremos en el servidor un Ubuntu Server usando una plantilla (Sin usar una ISO). Esta plantilla la descargaremos. Cuando esté instalado, instalaremos el Apache en él.
