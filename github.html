Zer da GIT?

Git bertsio-kontrol software bat da; hain zuzen, produktu edo konfigurazio batean egin daitezkeen aldaketak kudeatzeko programa da.

Zer da GitHub?

Talde-proiektuak antolatu eta egin ditzakezun plataforma da. Idatzizko kode guztia publikoki gordetzen da. Pribatua izateko, ordaindu beharko zenuke.

Configurar Git

Lo instalamos:

  sudo apt-get install git

Lo iniciamos:

  git init

Subir los cambios:

  git add .

Consolidar:

  git commit -m “NombreConsolidación”

Enlazamos el GitHub con nuestro correo:

  git config --global user.email pedrorodri983@gmail.com

Enlazamos la carpeta del git init con el repositorio de nuestro GitHub:

  git remote add origin https://github.com/Pedro2dw3/Wiki

Subimos los archivos:

  git push -u origin master

Para descargarnos los archivos ya subidos:

  git pull origin master

Otros comandos:

Descargar el repositorio entero:

  git clone https://github.com/Pedro2dw3/Wiki.git

Ver la consolidaciones hechas:

  git log

Ver si se ha hecho algún cambio:

  git status

Subir los cambios de solo un archivo:

  git add archivo.txt
