ZER EGIN DUGU?

Wiki bat egingo dugu. Bertan, erronka nola egin dugun eta zein programak erabili ditugun agertuko da.

Gu kontratatzen gaitun konpainia(irakasleak gure kasuan) bidaltzen du eta Proxmox nola erabiltzen den jakin nahi duen edozein pertzonari doa zuzenduta.

ZER ERABILI DUGU?

Momentuz, orrialde estatikoak izango dira: orrialde nagusia eta beste batzuk. Hauek HTML eta CSS hizkuntzetan sartuta egongo dira. Sublime Text erabiliz egingo ditugu.

Mini-zerbitzari batean Proxmox instalatuko dugu. Bertan, Ubuntu Server bat sartuko dugu. Gero, Ubuntu Server-ean Apache instalatuko dugu.

ORRIALDEAK

Hiru orrialde sortu ditugu: nagusia, Proxmox-rena, GitHub-rena eta Apache-rena. Guztiak beraien artean konektatuta daude esteken bidez.

Orrialde nagusian, erronkaren informazio orokorra agertzen da eta menu bat dago beste orrialdeetara joateko.

“GitHub” orrialdean, GitHub eta Git nola deskargatu eta instalatu ditu. Gainera, honekin egin dugun guztia agertzen. “Proxmox” orrialdean, Proxmox nola instalatu dugun eta honekin egin dugun guztia agertzen da.

“Apache” orrialdean, Apache nola instalatu dugun eta honekin egin dugun guztia agertzen da.

Bakoitzak, bere logoaren kolore nagusiak ditu.
