Zer da Apache?

Kode libreko web serbitzari bat da.

Instalatu

Apache zerbitzarian instalatu dugu hurrengo komandoarekin:

  apt-get install apache2

Gero, apache hasieratu dugu hurrengo komandoarekin:

  service /etc/init.d/apache2 start

Ondo hasieratu den ala ez frogatu dugu hurrengo komandoarekin:

  service /etc/init.d/apache2 status

Nola erabili dugu?

Sortu dugun index.html, beste orrialdeak, eta CSS-ak sartu ditugu honen /var/www/html karpetan zerbitzaritik ikusi ahal izateko.
